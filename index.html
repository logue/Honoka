KanataはHonokaを元に配色とフォントを変更したBootstrapテーマです

KanataはBootstrapテーマの一つですが、以下の特徴を持っています。

Easy to Start

KanataはBootstrapを元に製作されているため、非常に高い互換性を持っています。マークアップに関する規約はほとんど変わりません。

Replace Bootstrap

既にBootstrapを使って作成したウェブサイトがある場合は、CSSを置き換えるだけで簡単にKanataを利用できます。

Open Source

作成に使用したSASSファイルは全て公開されています。変数の定義ファイルを変更することで自分好みの設定に変更することも可能です。

Optimized Japanese

本家Bootstrapでは指定されていない日本語フォント（Noto Sans JP）に関する指定が行われているため、美しく日本語を表示できます。

Kanataは日本語を美しく表示します

本家Bootstrapでは指定されていない日本語のフォント指定を行っているので、日本語も美しく表示されます。
